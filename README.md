# Minimal Starter with Nodejs Webpack Karma Jasmien
## Get Started
```sh
npm run start
```
## Developement mode
* Terminal 1: ```npm run build```
* Wait for the build to finish
* Terminal 2: ```npm run server```

## Prod mode
Includes AoT 
```sh
npm run build:prod
npm run server
```
## Test
```sh
npm test
```
